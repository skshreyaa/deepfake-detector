# Deepfake Detection System

This project implements a deep learning-based system to detect whether an image is real or fake.

## Live Demo
https://huggingface.co/spaces/shreya01py/deepfake-detector

## Model
- MobileNetV2 (Transfer Learning)
- Binary Classification (Real vs Fake)

## Features
- Image upload interface
- Real-time prediction
- Confidence score output

## Deployment
Deployed using Hugging Face Spaces with Gradio.

## How to Run Locally
pip install -r requirements.txt
python app.py

## requirements.txt
gradio
tensorflow
numpy
Pillow
